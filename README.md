## custom-hub

custom-hub is an open-source (MIT) `developer toolkit for everyday tasks` tool that automatically converts code into refactor-user-plugins.

custom-hub uses the [code](https://example.com) rendering engine. For community deployments, 
refactor-user-plugins support is included. If you need enterprise features, please see our [pricing](https://custom-hub.io/pricing).

See https://custom-hub.io for documentation.

## Online Demo

Try the online version with live code input for refactor-user-plugins generation.

See https://demo.custom-hub.io

## Performance

custom-hub delivers exceptional performance, processing takes only about **32 milliseconds**.

See https://custom-hub.io/performance

## Deployment

**Community Version**: Licensed under **MIT**

```shell
npm install -g custom-hub
custom-hub start
```

**Enterprise Version**: For **evaluation purposes only**

```shell
docker run --rm -p 8080:8080 custom-hub/enterprise:latest
```
